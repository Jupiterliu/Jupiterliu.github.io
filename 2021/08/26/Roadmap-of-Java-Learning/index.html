Roadmap of Java Learning

发表于2021-08-26|更新于2021-09-01
|字数总计:393|阅读时长:1分钟|阅读量:

Basic Required Skills：

  1. Linux：
  2. Git & Github：
  3. HTTP/HTTPS：
  4. Data Structure & Algorithms：Linked list、arrays、binary tree、stack、queue、graph、hash table、DP（dynamic programming）
  5. SQL：
  6. Design Patterns（Tutorial）：three major categories：creational、structual、hehavioural

Step by Step Roadmap

  1. Prerequisties：
    • Java
    • Gradle || Maven：相比Mavne，Gradle结合了Maven的优点，已经成为现在的主流构建工具。Maven看XML文件中的dependencies依赖管理模块，TLS-Attacker中的为pom.xml文件（POM:Project Object Model-项目对象模型）；Maven的多模块架构，也即是面向服务的架构时，有parent模块，同时也有leaf子模块，遵从继承原则；
  2. CLI Tools：
    • Javac
    • Jar
    • Jstat
    • Jdeps
  3. Web Frameworks + Routers：
    • Spring
    • Play Frameworks
    • Spark：并行数据处理框架
    • Jersey
    • Spring Boot
  4. Databases：
  5. ORMs：“对象关系映射”框架（Java对象和关系数据库）
  6. Caching：缓存框架
  7. Logging：日志框架
  8. Real-Time Communication：Socket的Client和Server实时通信
  9. API Client：REST，提供给Java的API接口
  10. Application Server：Tomcat一类的服务
  11. Good to Know：Bean Vaidation（数据验证规范）、Bouncy Castle（算法库/API）、Gson（生成解析Json文件）、Apache Shiro（Java安全框架）
  12. Testing：测试，JUnit测试（TLS-Attacker中用的就是JUnit测试）
  13. Satic Code Analysis：Apache Yetus
  14. Task Scheduling：自动化任务执行
  15. Design Patterns：（learning）
    • Singleton：单例模式（Creational），存储数据时，不需要高并发访问资源时，避免死锁时
    • Factory：工厂模式（Creational），解决父类和子类的依赖问题，一般会出现ClassFactory类，就代表有工厂模式的利用
    • Observer：观察者模式（Behavioural），主体和多个观察者，观察者可以查看主体变化，只要变化主体就可以
  16. Microservises：微服务
